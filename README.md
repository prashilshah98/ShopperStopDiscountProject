# Shopper Stop Discount Project

This project demonstrates SOLID principles, Strategy Pattern, and progressive slab-wise discount calculation (similar to income tax slabs) for Shopper Stop customer types:

Silver
Gold
Platinum

## Includes:
-- Console application
-- Clean domain-driven architecture
-- Unit tests with xUnit + Moq
-- Code coverage using Coverlet

## Run the Console App

From solution root:

cd ShopperStopSolution/ShopperStopSolution
dotnet run


## Running Unit Tests

From solution root:
dotnet test

## Running Tests WITH Code Coverage
dotnet test --collect:"Code Coverage"
